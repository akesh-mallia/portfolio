Camp Randall Vibrations Research

Dynamic Structures and Data (DSD) Lab

2025-26

Background

“Jump Around” is an infamous tradition at Camp Randall Stadium that can cause noticeable vibrations throughout the bleachers. The stadium’s precise shape in response to these oscillations remains unknown. Previous tests in stadiums used accelerometers attached to points in the stadium. In this approach, data is only acquired at certain structural members in the stadium, and hypothetical cases cannot be tested. Instead, this study tests a 3D-printed model of the stadium on a shaker table. The printed model’s behavior is compared to data collected through cell phone accelerometers in the stadium during a football game. This project aims to use data processing to reveal the stadium's natural frequencies and mode shapes, improving knowledge of how Camp Randall Stadium responds to human-induced vibrations.

Challenges

Learning Curve

Starting as a freshman in Spring 2025, this project very quickly stretched my brain by introducing me to concepts like making Fourier transforms. Vibrations and non-linear dynamics can get very complicated quite quickly. I address this challenge by modeling the system in simpler ways (i.e, only a limited number of nodes in a finite element model). I also learn as quickly as I can through using my resources (lectures, professors to ask questions).

Results

Verified FEA model

I am confident the FEA model created in ANSYS for this project matches real-world behavior. This is because the axial acceleration data between the real world and simulation match in the frequency spectrum.

Presentation at the 2026 UW-Madison Undergraduate Symposium

On April 17, 2026, I presented a poster for this project at this symposium. It detailed my findings and spurred future directions for the project.
